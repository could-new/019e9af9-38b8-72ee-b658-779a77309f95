# Jarvis AI Assistant

A sleek, futuristic AI assistant chat bot interface inspired by J.A.R.V.I.S. Features a dark theme with neon cyan accents, a glowing status indicator, and a terminal-style typography layout.

## Features
- Futuristic dark UI with glowing accents
- Terminal/Courier typography style
- Simulated AI response processing
- Responsive layout for mobile and desktop

## Setup
Run the project using standard Flutter commands:
```bash
flutter pub get
flutter run
```
